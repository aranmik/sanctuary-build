# 성소판 21 — 사제 성장/재화 청사진 (PRIEST GROWTH ECONOMY BLUEPRINT)
**작성: 렌 · 2026-07-07 · EP15D 설계팩 · 상태: 렌 초안(유키 판정으로 확정)**

> **성소판은 5인 파티의 전장 상황을 읽고 수습하는 사제 게임이다. 플레이어는 공격하지 않는다. 플레이어는 파티의 리베로다.**
> **성장은 손을 대신해주지 않는다. 성장은 내가 읽은 상황에 더 좋은 답을 준비하게 해준다.**

- 관련: [09 스킬](./09_PRIEST_SKILL_BLUEPRINT.md) · [16 보상](./16_PROGRESSION_REWARD_BLUEPRINT.md)(보상=준비의 변화) · [19 저장](./19_SAVE_DATA_BLUEPRINT.md) · [18 범위](./18_CONTENT_SCOPE_BLUEPRINT.md)
- **이 문서는 구현 지시서가 아니라 장기 설계 기준서다.** 이번 카드에서 코드·수치·UI 변경 0. 태그: ✅확정 / 🔷후보 / ⏸️보류 / ⛔금지

## 1. 성장 철학 (✅ 잠금)

1. 사제 성장은 **자동 수습이 아니라 선택지 확장**이다.
2. 성장으로 사제가 공격자가 되면 안 된다. ⛔
3. 성장으로 정화/치유/보호 **판단**이 자동화되면 안 된다 — 판단은 영원히 플레이어의 몫. ⛔
4. 성장의 목적: **"상황을 읽은 플레이어에게 더 좋은 답을 주는 것."** 읽지 않은 플레이어에게 성장은 아무 일도 하지 않아야 한다.
5. 시험지(16 §1 상속): "이 성장을 받은 뒤, 성당/출정의 **선택**이 달라지는가?" — 달라지지 않으면 그것은 성장이 아니라 인플레다.

## 2. 금지 성장 (⛔ 전 기간)

자동 치유 · 자동 정화 · 자동 보호막 · 자동 차단 · 동료 자동 부활 · 사제 공격 스킬 · 단순 무한 스탯 상승 · 기본기/정화의 잠금(시작 6종+심장은 성장 이전에 항상 손에 있다, 16 §2).

## 3. 스킬 성장 단계 초안 (🔷 구조 · 구현 ⏸️)

**0단계 획득 → 1~3단계 성장 → 완성 특성 선택(배타 갈래)** 구조 후보.

| 단계 | 철학 |
|---|---|
| 0 획득 | 새 답이 손에 들어온다(16 스킬 해금과 동일 축) |
| 1~2 성장 | **상황 특화 소폭** — 숫자 상승 최소, "언제 더 좋아지는가"가 명확해야 함 |
| 3 성장 | 스킬 정체성 심화 |
| 완성 특성 | 2~3갈래 중 **하나만** 선택 — 성장의 끝은 강함이 아니라 **취향의 확정** |

- 첫 데모에서는 실제 성장 구현 ⏸️(보류 가능 — §8). 성장은 숫자 상승보다 상황 특화 중심 ✅.

| 기도씨앗(회복의 기원) | 예측 보상의 선명화 — §5 갈래 참조 |

## 5. 완성 특성 갈래 예시 (🔷 대표 7종 · 수치 없음 · 이름은 20 안전 지대 준수)

| 스킬 | 갈래 1 | 갈래 2 | 갈래 3 |
|---|---|---|---|
| 소생 | **오래울림** — 지속 시간 연장 | **첫숨** — 첫 틱 즉시/강화 | **마지막 약속** — 만료 시 소형 보호막 |
| 정화 | **빠른 손** — 쿨 감소 | **지운 자리** — 정화 대상 잠시 피해 감소 | **연계의 눈** — 정화→차단 성공 시 신성력 소량 |
| 기도씨앗 | **굵은 씨앗** — 발동 회복량 | **긴 잠** — 유지 시간 연장 | **넝쿨** — 전이 규칙 강화(횟수/범위) |
| 보호막 | **두꺼운 벽** — 흡수량 | **짧은 숨** — 재적용 잠금 완화 | **강타 대비** — 강타 피해 특화 흡수 |
| 성심 집중 | **깊은 고요** — 대형 스킬 특화 | **잦은 고요** — 쿨 감소 | **고요의 잔향** — 사용 후 소량 마나 회복 |
| 공명의 기도 | **넓은 울림** — 대상 수 | **겹울림** — 낮은 HP 가중 | **빠른 울림** — 광역 예고 창 특화 시전 |
| 긴급 구원 | **구원의 여운** — 구원 대상 잠시 피해 감소 | **재기** — 구원 직후 대상 행동 복귀 강화 | **충전 감각** — 신성력 수급 리듬 소폭 |

- 규칙: 갈래는 **배타 선택** · 어느 갈래도 §2 금지(자동화)로 넘어가면 안 됨 · 모든 이름은 [20 §2](./20_NAMING_LEGAL_SAFETY_BLUEPRINT.md) 안전 지대 조어.

## 6. 성장 재화 (🔷 · 첫 데모 2종 이하 ✅)

| 재화 | 용도 | 상태 |
|---|---|---|
| **기도의 잔향** | 일반 성장 재화(스킬 1~3단계) | 🔷 데모 최소 후보 |
| **성소의 인장** | 보스 최초 클리어 증표 — 스킬 해금/완성 특성 개방 | 🔷 데모 최소 후보 |
| 정화의 파편 · 수호의 촛불 · 기억의 씨앗 | 장기 세분화(계열 특화 재화) | ⏸️ 장기 |

- 명명 주의: '기도의 잔향'과 스킬 '빛의 잔향'의 '잔향' 중복은 결이 달라 허용 판단 — 최종은 Store Naming Audit(20 §5)에서 재점검 ⏸️.

## 7. 재화를 얻는 행위 (🔷) — **이미 세고 있는 값과 1:1**

| 행위 | 근거 값(코어 report()/어댑터 — 전부 기구현) |
|---|---|
| 보스 클리어 | `r.r==='victory_kill'` |
| 생존 승리 | `r.r==='victory_survive'` |
| 리포트 힌트 목표 달성 | 직전 판 힌트 조건의 이번 판 해소(어댑터 비교 🔷) |
| 정화→차단 연계 성공 | `r.ints[].asst` |
| 폭발 무효 | `r.bombDef===r.bombTot` (bombTot>0) |
| 광역 후 복구 성공 | `r.care` |
| 마나 고갈 없이 클리어 | `r.oomT===0` |
| 사망자 0 | `r.deaths.length===0` |

- **주의(✅)**: 재화 획득은 특정 플레이를 강제하지 않는다 — "잘 읽은 전투"를 부드럽게 보상한다. 조건 미달의 벌점 ⛔ · 반복 강요형 파밍 ⛔(16 §4).
- 위 표의 의미: 재화 훅은 **코어 무접촉**으로 연결 가능하다(값이 이미 존재) — 미래 카드의 착공 근거.
