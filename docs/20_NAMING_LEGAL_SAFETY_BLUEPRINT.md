## 1. 이름 3층 체계 (✅ 유키 채택)

| 층 | 노출 범위 | 규칙 |
|---|---|---|
| **성소판 표시명** | 출시 UI·스토어·마케팅의 **유일한** 노출층 | 성소판 고유명 |
| **현재 UI명** | 개명 전 과도기의 빌드 문자열 | UI Rename 카드로만 일괄 변경(오늘 ⛔) |
| **originRef** | 문서 내부 전용 | UI·스토어·마케팅 노출 ⛔ |

- 문서 표기 규약(✅): 최초 1회 "성소판명(originRef)" 병기, 이후 성소판명 단독 가능. 코드 키(`heal`/`renew`/`seed`…)는 내부 식별자로 자유.

## 2. 이름 안전 3계층 (✅ 내부 관례)

| 계층 | 정의 | 판단 |
|---|---|---|
| **A. 일반어 지대** | 사전 단어·일반 조합 — 소생·소실·정화·보호막·치유·기도·전사·도적·마법사·주술사·사제 | 안전 — 타 게임 한글판과 겹쳐도 일반명사는 특정 작품에 귀속되지 않는다는 내부 관례 |
| **B. 조어 주의 지대** | 특징적 코인드 프레이즈·콜론 형식 — "신의 권능: 보호막", "회복의 기원", "내면의 집중력" 류 | **그대로 사용 ⛔** — 성소판 고유명으로 우회(이미 실행: 기도씨앗·성심 집중·수호 서약 등) |
| **C. 고유명 금지 지대** | 세계관 인명/지명/종족/보스명·로고·표기 포맷 | ⛔ 절대 금지 |
