# 성소판 16 — 성장/보상 청사진 (PROGRESSION REWARD BLUEPRINT)
**작성: 렌 · 2026-07-07 · EP09 설계팩 2차 4/8 · 상태: 렌 초안(유키 판정으로 확정)**

> **성소판은 5인 파티의 전장 상황을 읽고 수습하는 사제 게임이다. 플레이어는 공격하지 않는다. 플레이어는 파티의 리베로다.**
> **보상은 수습 손맛을 대체하지 않는다. 보상은 "다음 준비가 달라지는 것"이다.**

- 관련: [08 플로우 §1 루프 7단계](./08_GAME_FLOW_BLUEPRINT.md) · [11 마을 §5 유물](./11_VILLAGE_BLUEPRINT.md) · [19 저장](./19_SAVE_DATA_BLUEPRINT.md)
- 태그: ✅확정 / 🔷후보 / ⏸️보류 / ⛔금지 · 코드·수치 변경 없음

---

## 1. 대원칙 (✅ 잠금)

1. **손맛 불가침.** 보상이 없어도 전투가 재미있어야 하고, 보상은 그 재미를 잠식하지 않는다. 보상으로만 이길 수 있는 판 ⛔.
2. **보상 = 준비의 변화.** 좋은 보상의 시험지: "이 보상을 받은 뒤, 성당/길드에서의 선택이 달라지는가?"
3. **선택 방식 변경 > 수치 인플레.** +10%보다 "다르게"가 우선(11 §5 유물 원칙과 동일 혈통).

## 3. 진행 구조 (🔷)

- **1차 데모 = 선형 해금**: 모르가스 → 셸 보스 1 → 셸 보스 2. 범위 통제·서사 명확. 병렬 선택 트리는 장기 후보 ⏸️.
- 해금 단위는 "클리어"가 기본. 조건부 해금(무사망 등)은 ⏸️ — 도전 과제화는 손맛 검증 후.

## 4. 보상 리듬 (✅ 원칙)

- **클리어 = 해금(새 질문/새 답) · 리포트 = 지식 · 반복 = 숙련.** 반복 플레이의 보상은 손맛과 기록 갱신이지 숫자가 아니다 — **파밍 루프 ⛔.**
- 실패도 보상한다: 실패 리포트 문장(15 §5)이 열쇠를 준다 — "죽어도 배운다"가 성소판의 경제.

## 5. 금지 (⛔)

수치 인플레 우선 강화 · 정화/기본 6종 해금 잠금 · 파밍 루프 · 보상의 손맛 대체 · 저장 정책(19) 확정 전 진행도 구현 · 반복 강요형 게이트(같은 보스 N회 클리어 조건).
