## 3. Visual Battle Language (14 문서와 공유 어휘)

- 선 어휘·우선순위(차단선 > 단일 대상선 > 개입선 > 링, 광역 틴트=배경 공존)·동시성(핵심선 1~2, 보스 단일 시전이 구조 보장)은 **14 §3이 원본** — 여기서는 UI 규칙만 추가:
  1. **선은 정보다.** 정보가 끝나면 선도 끝난다(강타선=시전 중만, 개입선=0.46s) ✅.
  2. 선의 색은 칩·콜아웃과 **같은 색 언어**를 쓴다 — 위험=적 · 회복=녹 · 보호=청 · 신성=금 · 해제=자 ✅.
  3. 신규 선 추가는 기존 선 1개 강등과 교환 🔷(1-in-1-out — §5).

## 5. 정보 예산 (🔷 규칙 제안)

- **전투 화면 읽기 예산 = 0.5초.** 새 요소(칩 종류·선·배지 상태·게이지)를 추가하려면 기존 요소 하나를 제거·강등·통합한다(**1-in-1-out**).
- 같은 정보의 이중 절규 금지 🔷 — 예: 강타 예고가 경고바+시전바+선 3중이면, 층별 역할 분담(멀리→가까이)만 남기고 중복 문구 제거.
- 예산의 심판은 나라 폰 실기(상위 0.1% 보정 상시 적용 — 나라가 "빠듯"하면 일반 플레이어는 "불가").
