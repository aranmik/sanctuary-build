## 1. 동료 공통 원칙 (✅ — 10 문서 상속·재잠금)

동료는 직접 조작하지 않는다 ⛔ · 동료는 사제의 손을 대신하지 않는다 ⛔ · 동료는 자동으로 전투를 해결하지 않는다 ⛔ · **동료는 기회와 사고를 만든다** ✅ · **힐러는 사제뿐이다** ✅ · 동료 힐러화 ⛔(자동 치유·스마트 힐·전투 부활 포함).

## 2. 7종 로스터 (잠정 확정 🔷 — 구현은 전부 ⏸️, 기구현 4종만 ✅)

| # | 동료 | 역할 | 한 줄 정체성 | 상태 |
|---|---|---|---|---|
| 1 | 전사 | 메인 탱커 | **"내가 맞는다"** — 보스 정면의 기준점 | ✅ 구현 |
| 2 | 기사 | 보조 탱커 | **"남을 대신 막는다"** — 사고 방지 전문 | ⏸️ 셸 |
| 3 | 도적 | 딜러+차단 | 정화가 살리는 차단수 | ✅ 구현 |
| 4 | 마법사 | 딜러 | 유리몸 고화력 — 살리면 시간이 줄어든다 | ✅ 구현 |
| 5 | 궁수 | 딜러+예고/표식 | 화력은 낮고 정보 가치가 높다 | ⏸️ 셸 |
| 6 | 주술사 | 아군 전투 지원 | **우리 편의 좋은 시간을 만든다** | ✅ 구현 |
| 7 | 흑마법사 | 적 약화/디버퍼 | **적의 나쁜 상태를 만든다** | ⏸️ 셸 |

## 3. 전사 vs 기사 (✅ 구분 잠금)

| | 전사 | 기사 |
|---|---|---|
| 핵심 | **내가 맞는다** | **남을 대신 막는다** |
| 일 | 보스 정면 고정 · 평타/강타/전열 붕괴를 받아냄 | 특정 사고를 대신 막거나 피해를 나눔 |
| 약점 | 죽으면 파티가 흔들림(전열 승계 — 배지가 이미 증명) | 메인 탱킹은 전사보다 약함 |
| 사제 연계 | 깊은 치유·소생의 기본 캔버스 | **보호막/소생/수호 서약과 연계** — "대신 막는 순간"이 사제 개입 창 |

## 4. 도적/마법사/궁수 — 딜러의 기회비용 (✅ 정체성)

- **도적**: 딜러+차단. **디버프가 있으면 차단 실패** → 정화해주면 차단 기회가 열린다(심장 연계 ✅ 기구현).
- **마법사**: 고화력·유리몸. **살리면 전투 시간이 줄어든다** — 대신 사제 자원을 많이 요구. "마법사를 살리는 비용"이 곧 화력형 조합의 가격표.
- **궁수**: 안정 원거리. 순수 화력은 마법사보다 낮지만 **위험 예고/표식/약점 포착** — 정보 가치. ⚠️ 표식은 기존 예고 계층(경고바/시전바/배지)에 **얹지 말고 통합**해야 함 — 17 §5 정보 예산 심사 필수 ⏸️.

## 5. 주술사 vs 흑마법사 (✅ 대비 잠금)

| | 주술사 | 흑마법사 |
|---|---|---|
| 방향 | **아군 생존선을 올린다** | **적 피해선을 낮춘다** |
| 수단 | 방어 지원·전투 의지·지원 창 | 공격력 감소·약화·디버프 |
| 성질 | 파티가 안정적일수록 효율 증가 | 적을 약하게 만들되 본인 리스크/유지 부담 존재 가능 |
| 한 줄 | 우리 편의 좋은 시간을 만든다 | 적의 나쁜 상태를 만든다 |

- **약화의 가드레일(✅)**: 흑마법사의 약화는 "불가능을 가능으로" 만들되 **긴장을 제거하면 안 된다** — 약화 후에도 15 문서의 보스 질문이 성립해야 한다. 질문이 사라지는 약화 = 난이도 삭제 ⛔.

## 6. 보스 압력 × 동료 해법 매트릭스 (🔷 — 나라님 핵심 예시 반영)

| 압력(예시) | 해법 | 없을 때 |
|---|---|---|
| **보스 A — 공격력 100** | 주술사의 아군 방어 증가로 버틸 수 있음 | 사제 치유 부담 급증 — **불가능이 아니라 손이 매우 바빠짐** |
| **보스 B — 공격력 180** | 주술사 방어만으론 부족 → **흑마법사의 적 공격력 50% 감소 필요** | 수호 서약+긴급 구원 총동원 — 가능하되 손이 불탐 |
| **보스 C — 공격력 220** | **흑마법사 약화 + 주술사 방어가 동시에 필요** | 하나라도 없으면 "읽을 시간" 자체가 사라짐 |

**정리(✅)**: 어떤 상황에서는 특정 동료가 유리하다 · 어떤 상황에서는 두 동료를 같이 써야 한다 · **동료 선택에서부터 유저의 강함 증명이 시작된다.**
**주의(✅)**: "없으면 불가능"보다 **"없으면 사제 손이 매우 바빠짐"을 우선**한다. 좋은 조합은 전투를 자동 해결하지 않고, **위험을 읽을 시간과 수습 여지를 넓힌다.**

### 6-1. 태그 ↔ 동료 해법 연결 (🔷 — 08 태그 어휘와 크로스)
⚡ 차단 필요→도적 · 🔨 연속 폭사→기사(대신 막기)+수호 서약 · 🩸 다중 낙인→흑마법사 약화·대정화 · ⏳ 장기 마나전→주술사 창+성심 집중 · 👁️ 사제 주시(게이트 ⏸️)→기사 대신 막기 후보. → 장기적으로 출정 확인 경고의 어휘가 된다(§10).

## 7. 4인 조합 예시 (🔷 — 장점/빠지는 해법/위험)

| 조합 | 장점 | 빠지는 해법 | 위험 |
|---|---|---|---|
| **안정형** 전사+기사+도적+주술사 | 이중 탱킹·차단·지원 창 | 화력·원거리 정보 | 처치 승리가 어려워 75초 생존 의존 — 광폭화 구간 부담 |
| **화력형** 전사+도적+마법사+궁수 | 최속 처치·차단 확보·정보 | 지원/약화 전무 | 유리몸 관리로 사제 자원 폭식 — 사고 시 복구 수단 없음 |
| **정보형** 전사+도적+궁수+주술사 | 예고 이중화·차단·창 | 순수 화력 | 정보 과잉 — 17 예산 심사 필요 |
| **저주형** 전사+마법사+흑마법사+주술사 | 압력 자체를 낮춤·창 | **차단 부재(도적 없음)** | ⚡ 태그 보스 상대 불가급 — 출정 확인 경고 1순위 후보 |
| **수비형** 전사+기사+궁수+주술사 | 최대 생존선·예고 | 차단·화력 | 장기전 강제 → ⏳ 마나 압박 보스와 최악 궁합 |

- 발견(✅ 원칙화): **모든 조합은 새 답과 함께 새 구멍을 산다** — 조합의 구멍이 곧 그 판의 질문. 구멍 없는 만능 조합 ⛔.

## 8. 동료 성장 방향 (🔷 — 원칙 ✅)

**원칙**: 동료 성장은 **사제 개입의 가치를 키운다** · 동료가 스스로 모든 문제를 해결하게 만들면 ⛔ · 성장은 보스 질문에 대한 해법 조각을 더 선명하게 한다.

| 동료 | 성장 방향 예 |
|---|---|
| 도적 | 정화 후 차단 반응 향상(연계 창 확대 — 사제가 만든 기회가 더 확실히 회수됨) |
| 기사 | 대신 막기 조건/효율 개선 |
| 궁수 | 예고/표식 정확도 향상 |
| 주술사 | 아군 지원 창 강화 |
| 흑마법사 | 적 약화 유지/효율 강화 |
