> **핵심 문장 3줄**
> 1. 1차 데모는 분량이 아니라 **문법의 증명**이다 — 서로 다른 질문 세 개가 서로 다르게 느껴지면 성립한다(18 상속).
> 2. 완료의 반대는 미완이 아니라 **혼동**이다 — 이 문서는 "무엇을 만들면 데모인가"의 경계선이다.
> 3. 남은 길은 여섯 걸음이고, 그중 두 걸음(파쇄자·심연 실전투)은 **하나의 문(Boss Core Gate) 뒤**에 있다.

## 1. 1차 데모의 한 문장 정의 (✅ 잠금 후보)

> **성소판 1차 데모는, 5인 파티의 전장 상황을 읽고 수습하는 사제의 손맛이 서로 다른 세 개의 보스 질문 앞에서 성립함을 폰 실기로 증명하는 빌드다.**

- 보존 의미 5요소(불변): **사제 게임 · 전장 상황 읽기 · 수습 · 3개의 보스 질문 · 손맛 검증** — 문장을 다듬어도 이 다섯이 빠지면 무효.
- 세 질문(24·17B 확정): 모르가스 "너는 순서를 읽는가?"(정화/차단) · 강철의 파쇄자 "네 마나는 나의 파괴보다 오래 버티는가?"(강타/버티기) · 갈증의 심연 "무엇을 안 쓸 것인가?"(마나 압박/절약 운영).

|---|---|

## 5. 1차 데모 HOLD 조건 (✅ — 신호 → 행선지 카드 맵)

HOLD는 실패가 아니라 **다음 카드의 씨앗**이다(97 상속). 신호가 오면 아래 행선지로:

|---|---|
| 정보가 많아 무엇을 해야 할지 모름 | Battle Screen Density Audit 01(25) |
| 스킬 버튼이 거짓말함(상태↔게이트 불일치) | Skill Tile Visual Contract 01(25) — "거짓말 금지 구역" 위반 |
| 예보/결과 FX가 헷갈림 | FX Contract Audit 01(26 — 시제 문법 위반) |
| 정화 이유가 안 읽힘 | Cleanse→Rogue Interrupt VBL Polish 01(26) |
| 성심 집중이 그냥 무료 버튼으로 느껴짐 | Sacred Focus FX Polish 01 + 수치 HF(쿨/유지 — 유키) |
| 로드아웃 선택이 무의미함 | 로드아웃 HF(07 계보) + 보스별 요구 로드아웃 점검(15 §4 네 고리) |
| 리포트가 다음 행동으로 이어지지 않음 | Report Readability Polish 01(25) / 힌트 규칙 HF |
| 모바일에서 터치/시야가 불편함 | 07-HF 시인성 + Density Audit |
| **신규 보스 추가로 Morgas 손맛이 깨짐** | 즉시 전면 HOLD — 보스 밸런스는 게이트 재-baseline 의식과 세트로 재검토 |
