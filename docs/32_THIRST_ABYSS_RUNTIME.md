## 2. 갈증의 심연 전투 질문 & 마나 압박 구조

**새 어휘 `drain`(신규 1개 — 사유·영향 명시):**
- **왜 필요했나**: "마나가 천천히 말라간다 / 무엇을 안 쓸 것인가"를 정직하게 만들려면 **사제 마나를 직접 압박하는 이벤트**가 필요. 기존 어휘(smash=탱 피해·brand/bomb/judge=정화 대상·valor/enrage=버프)는 전부 "피해" 문법이라 마나 압박을 표현할 수 없다.
- **효과**: ① 사제 마나 흡수(62 · 광폭화 95) — `Math.max(0,...)` 클램프로 **음수·억울 방지**(마나를 갑자기 0으로 만들지 않음). ② 파티+사제 약한 광역(42 · 광폭화 64) — "힐로 반응하고 싶은 유혹"을 만들고, **반응하면 마나가 이중으로 빠지는 딜레마** = "무엇을 안 쓸 것인가" 그 자체.
- **영향 범위**: fireScript에 `case 'drain'` 1개 추가 + nextPattern NM 1항. **기존 마나 구조(payMana/regen/mana 필드) 무접촉** — mana를 읽고 깎기만 한다. 모르가스/파쇄자 SCRIPT엔 drain이 없으므로 두 보스 전투 경로에 `drain` case는 도달 불가 = 회귀 0.
- **ABYSS_SCRIPT(14이벤트)**: drain×11 + valor×2(t16/42 — 마나 숨통) + enrage(t56 — 후반 압박 강화). smash/brand/bomb/judge **0**(강타·정화·차단 아님).
- **수치**: HP **8400**(파티 이론 최대 화력 ≈8176보다 높게 — 처치는 예외, **기본은 75초 생존/버티기**) · melee 50/2.2s(광폭 66/1.9) · offTankMul 1.4(탱 흘림 덜 중요).

**승리 모양 = 생존 버티기**: HP를 이론 화력보다 높게 잡아, 활성 플레이는 대개 **75초 생존승**으로 수렴한다. 파쇄자(처치 레이스)와 정반대 — 심연은 "때려눕히는" 보스가 아니라 "**마나가 마르기 전에 버텨내는**" 보스다.

## 3. 성심 집중 검증 의도 (현재 범위만 사용)

- 성심 집중(p13 · CD30/armed8/GCD)의 **기존 범위 그대로** — A안 확장·대상 스킬 추가·UI 대공사 **0**. 환급 {빠른 치유·보호막·정화} + 진무료 {소생·기도씨앗} + 제외 {깊은 치유·공명의 기도·긴급 구원} 불변.
- 심연에서의 의미: 마나가 계속 빠지는 전투라 **"다음 힐을 언제 무료로 쓸까"**가 곧 자원 운영의 핵심. drain 압박 구간에 armed를 맞춰 켜면 숨통이 트인다.
- ★**"쓰고 싶다"는 되지만 "없으면 답이 없다"는 아니다**: 성심 집중 없는 로드아웃(빠른 치유·소생·기도씨앗의 저비용 운영)으로도 버틸 수 있게 설계. 헤드리스 봇도 focus 유무 양쪽 다 생존승 도달(focus는 OOM 시점을 늦춰 숨통을 확인).

- **보호막 0 · 소생 1512(74%)** = 파쇄자(보호막 흡수 2808)와 **정반대 손**(지속힐 위주)을 자연스럽게 선택 — "강타가 없으니 보호막 대신 꾸준한 힐"을 몸으로 골랐다.
- **정화 0회** = 정화/차단 보스 아님이 손맛으로 증명. **오버힐 8%** = 힐 난사 없이 절약 운영 성립("무엇을 안 쓸 것인가" 실천).
- **마나 45초에 바닥** = 마나 압박이 실제로 작동, 후반은 관리 싸움(상위 0.1% 보정 시 일반 유저는 더 이르게 바닥). **HP 5% 잔존 생존승** = 처치 직전까지 갔으나 못 죽이고 버텨 이김 = **생존 버티기형 설계 정확 작동**.
- 나라님 체감: **"신선했다, 재미있다."**

**정체성 확정(잠금)**: 갈증의 심연 = **마나 압박 / 절약 운영 / 뒤늦게 비용이 보이는 보스**. 처음엔 평범해 보이지만 중반부터 마나가 바닥나며 자신의 낭비를 깨닫게 만든다. 마나 고갈은 **억울한 실패가 아니라 마지막까지 버티는 긴장**으로 작동.
