## 8. 도적 최종안과 이유 (안 C)

- **안 C 채택**: 먹빛 검정(#35323f 두건/#2b2834 몸)으로 무채색 척후. 단검 은회(#e2e6f0)로 대비 확보. 킥은 냉회색(#635d6a). 보라는 스카프 표식(#6f5f97)에만 잔향. → "검정 계열의 척후, 보라는 흔적만 남은" 인상. B(완전 무채)는 보라 정체성이 아예 사라져 도적의 원래 색 기억과 단절되고, A는 아직 흑보라라 마법사 보라와 근접. C가 검정 정체성 + 최소 연결 + 마법사와 최대 분리.

## 9. 마법사 기존 상태 분석 (before = Preview 02)

- 로브 남색(#343150)·배 띠 회보라(cream #d9cdec). 모자·구체·별은 보라/노랑. → 배/중앙 남색이 도적 검정과 명도·색온도가 겹쳐 뭉갬. 노랑(gold) 정체성이 별·구체에만 있어 약함.

## 11. 마법사 최종안과 이유 (안 C)

- **안 C 채택**: 로브를 남보라(#3e3563)로 올려(도적 무채색 검정과 색상 분리) + 배 띠를 금색(#e5c05f)으로. 모자 자주·구체·별 노랑과 함께 **보라+노랑 정체성이 몸 전체로 확산**. A는 분리는 되나 노랑이 약하고, B 자주 띠는 보라 과잉으로 노랑 정체성이 밀림. C가 "보라와 노랑의 시전자"를 가장 선명하게 하고 도적 무채색과 최대 대비.

## 12. before/after 차이

- **도적**: 보라 두건/몸 → 먹빛 검정 몸 + 은회 단검 + 냉회 킥 + 스카프 보라 잔향. 무채색 그림자 척후로 하강.
- **마법사**: 남색 배 → 남보라 로브 + 금 배 띠. 보라·노랑이 몸 전체로 상승.
- 두 방향(도적 ↓무채색 / 마법사 ↑보라노랑)이 반대로 움직여 **중간톤 겹침이 양쪽에서 해소**.

## 13. 4인 파티에서의 인상 변화

- before 파티: 전사(파랑)·도적(보라)·마법사(보라)·주술사(청록) — 도적·마법사가 보라로 근접해 둘이 헷갈림.
- after 파티: 도적(검정)·마법사(보라+금) — 무채색 척후 vs 보라노랑 시전자로 **즉시 구분**. 전사·주술사 무변경이라 파티 색 스펙트럼(파랑·검정·보라금·청록)이 오히려 더 넓고 선명. 전대물 인상 없이 각자 다른 옷감의 모험 파티.

## 14. 같은 세계감 유지 근거

- 조형 실루엣·파츠 구조·얼굴(점눈+볼터치)·외곽선·접지 그림자/오라 규칙 전부 Preview 02 그대로. 바뀐 것은 `--sb-r-*`/`--sb-m-*` 색 변수뿐. color-mix로 하이라이트/섀도를 main·cloth에서 자동 파생시켜, 색만 바꿔도 같은 재질 문법(같은 손이 지은 느낌)이 유지된다.

## 15. 전대물 인상 악화/완화 여부

- **완화**. Preview 02가 "중립 몸통으로 전대물 완화"였다면, 이번엔 도적을 무채색으로 더 내리고 마법사를 보라노랑으로 더 올려 **색 역할이 더 뚜렷**해졌다. 4인이 각기 다른 색 정체성(파랑/검정/보라금/청록)이라 "색깔 전대"가 아니라 "역할이 다른 분대".

## 16. 귀여움 유지 여부

- 얼굴 살구톤(#ffd9b8) 유지로 검은 두건에서도 표정이 뜬다. 둥근 파츠·부드러운 하이라이트(color-mix +흰 15~20%)로 검정이 칙칙하지 않고 "먹빛 벨벳" 느낌. 마법사 금 띠는 작은 포인트라 과하지 않다.

## 17. 전사/주술사 무변경 또는 최소 비교 확인

- 전사·주술사 색 변수(`--sb-w-*`/`--sb-s-*`) **완전 무변경**. 4인 파티 비교(C 섹션)에서 비교 기준으로만 등장. 손대지 않음.

## 18. `sb-` 네임스페이스 확인

- 신규 클래스는 전부 `sb-`(피규어) + `pol-`(색안 오버라이드) 접두. 일반명 0.

## 19. 3층 transform 유지 확인

- `.sb-unit`/`.sb-react`/`.sb-fit`/`.sb-fig` 유지. 색 조정은 파츠 배경(변수)만 건드리고 transform 레이어 무관.
