A. 🗡️ 도적 — 검정 계열 색안 비교 (before → A/B/C)

before = Preview 02(보라 메인). A 차콜+흑보라 / B 강한 검정+은회 단검(보라 제거) / C 검정 메인 + 미약한 보라 표식·냉회 킥(★채택). 얼굴은 살구톤 유지라 검은 두건에서도 안 묻힘.

B. 🔮 마법사 — 배/중앙 분리 색안 비교 (before → A/B/C)

배 부위(중앙 띠)가 도적 검정과 겹치던 문제. A 배 남보라 / B 배 자주 띠 / C 배 금 띠(보라+노랑 강화·★채택). 모자·구체·별의 보라/노랑은 유지.

C. 🎭 4인 파티 — before vs after (채택안)

before · 현행(Preview 02)
after · 도적 C + 마법사 C

전사·주술사는 무변경(비교 기준). after에서 도적이 무채색 검정으로 내려가고 마법사 배가 금빛으로 올라와 둘의 중간톤이 분리. 같은 세계감(얼굴·외곽선·접지 규칙)은 유지.

D. 🎬 대표 포즈에서의 색 (채택안)

도적 기본/차단 준비, 마법사 기본/충전 — 포즈가 바뀌어도 검정 척후·보라노랑 시전자 정체성이 유지되는지.

E. ⭐ 최종 추천안

🗡 도적 · 안 C
먹빛 검정 두건/몸 + 은회 단검 + 냉회 킥. 보라는 스카프 표식만. "검정 계열의 척후, 보라는 흔적"
🔮 마법사 · 안 C
남보라 로브 + 배 금 띠 + 자주 모자 + 노랑 구체/별. "보라와 노랑의 시전자". 도적 무채색과 분리
